A propos de moi

Booki

Ce projet portait sur la création de la page d'accueil du site d'une agence de voyage. En utilisant HTML et CSS, j'ai travaillé à partir d'une maquette Figma pour développer le site aussi bien pour ordinateur que pour mobile, en adoptant une approche responsive.
Les objectifs pédagogique de ce projet étaient les suivants :

  • Implémenter une interface responsive avec HTML et CSS
  • Installer un environnement de développement front-end

Oh My Food

Ce projet portait sur l'interface mobile-first du site d’une start-up. En utilisant Sass, j'ai implémenté des animations CSS et versionné le projet avec Git et GitHub. Mon principal défi était d'intégrer une maquette mobile-first à partir de maquettes disponibles pour mobile et desktop, ainsi que d'un prototype sur Figma.
Les objectifs pédagogiques de ce projet étaient les suivants :

  • Intégrer une maquette en mobile-first pour comprendre l'importance des sites web adaptatifs et réactifs.
  • Mettre en œuvre des animations CSS pour enrichir l'expérience utilisateur, rendant la navigation plus interactive et engageante.
  • Utiliser Sass, un pré-processeur CSS, pour écrire des styles plus organisés et maintenables.
  • Versionner le projet avec Git et GitHub, se familiarisant ainsi avec des outils essentiels pour la gestion et la collaboration sur des projets de développement web.

Print It

Ce projet portait sur la dynamisation du site internet statique d'une imprimerie en y ajoutant un carrousel interactif. En utilisant JavaScript, j'ai exploré les fondamentaux de ce langage pour rendre le site plus dynamique et engageant. À partir du code HTML existant, j'ai intégré un carrousel en JavaScript, ce qui m'a permis de comprendre comment JavaScript interagit avec HTML et CSS pour créer des éléments web interactifs.
Les objectifs pédagogiques de ce projet étaient les suivants :

  • Programmer en utilisant les fondamentaux de JavaScript
  • Intégrer des fonctionnalités interactives telles qu'un carrousel
